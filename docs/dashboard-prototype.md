# Dashboard prototype outcome

## Question

Which visual hierarchy should the annual Progressio dashboard use before production development starts?

## Decision

The selected direction is **Synthèse par niveau**:

- show annual indicators first;
- group class summaries by level;
- make the class progression the primary next destination from each level;
- keep advancement and tracking follow-up visually distinct.

The priority-first and annual-horizon variants were exploratory alternatives and are not part of the selected direction.

## Status

Validated from the throwaway UI prototype on 2026-07-13. The prototype code was removed; the production implementation must be written separately.
